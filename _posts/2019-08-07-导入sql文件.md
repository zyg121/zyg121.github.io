共有两套方案导入文件

 ##方案一
 

登录数据库
mysql -u root -p123
检查已经存在的数据库
show databases;
创建新的数据库
create database test;
\# 查看已经存在的是为了避免重复创建
选择你所创建的数据库
use test;
导入sql文件
source db-test.sql;
\#注意sql文件的路径
查看导入结果
show tables;
退出数据库
exit;

##方案二
 
mysql -u root -p123 test < test.sql
\# mysql -u 用户名 -p 密码 数据库名 < 数据库名.sql
